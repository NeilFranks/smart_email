This service is a school project and should not be used by any person besides those involved in the project. This is strictly for learning purposes.

This service stores your the GMail credentials you give it access to. It uses the credentials to retrieve and read email data from GMail, and has the ability to write drafts, send emails, and all other functionalities provided by Google's GMail API.

For informatin on how Google stores and uses your data, see here: https://policies.google.com/technologies/partner-sites

For a list of functions that the API allows this service to do with your credentials, see here: https://developers.google.com/apis-explorer/?hl=en_US#p/gmail/v1/
